# Death Road 2 Canada python trainer by XiAnzheng
More info read the Update Log 

## Feature 
* Unlimited Health (Not Working yet on Special Character)
* Unlimited Loot
* Unlimited Ammo (Not working yet on Special or Throwing or Spawn Ammo Type)
* Max Stats (Not Working yet on Special and Animal Character)
* Inventory Editor 
* item Quantity Editor
* Can be use on Modded DR2C (Depends how the Mod change the game)

### Module Used 
* ReadWriteMemory
* CustomTkinter

### How to use?
Just run the app.py 
or compile it with anything you like CX_Freeze , Nuitka , PyInstaller
Example iam using Nuitka:
```bash
python -m nuitka --windows-console-mode=disable --onefile --windows-uac-admin --enable-plugin=tk-inter --remove-output app.py
```
